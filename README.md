# 🚴‍♂️ Bike Sales Dashboard – Excel Data Analytics Project

## 📘 Overview
The **Bike Sales Dashboard** is a data analytics project built in **Microsoft Excel** to explore and visualize customer purchasing behavior.  
The goal was to identify how **demographic and lifestyle factors**—such as income, age, education, and commute distance—affect the likelihood of **purchasing a bike**.

This project demonstrates strong skills in **data cleaning, transformation, pivot analysis, and dashboard design** to extract actionable business insights.

## 🔄 Project Stages

### 1. **Data Preparation**
- Imported the raw customer dataset into Excel.
- Reviewed all fields for completeness and consistency.

### 2. **Data Cleaning**
- Removed duplicate entries to ensure data accuracy.
- Corrected inconsistent capitalization and formatting (e.g., gender, region).
- Handled missing or incorrect values by applying logical corrections.
- Verified data types (numeric vs. text) for proper pivot table calculations.

### 3. **Data Transformation**
- Created a new calculated column **“Age Bracket”** to categorize customers:
  - **Adolescent:** Below  31  
  - **Middle Age:** 31–54  
  - **Old:**  55 and above  
- Standardized commute distance ranges for easier grouping.  
- Ensured income values were properly formatted as currency.

### 4. **Data Analysis**
- Built **pivot tables** to summarize relationships between:
  - Income and bike purchase status
  - Age bracket and purchase rate
  - Commute distance and region-based trends  
- Derived insights on which demographic factors most influenced bike sales.

### 5. **Dashboard Creation**
- Designed an **interactive Excel dashboard** using:
  - Pivot Charts
  - Slicers (Marital Status, Region, Education)
  - Clean and consistent layout with color-coded visuals
- Optimized chart titles, axis labels, and formatting for presentation.

## 📊 Dataset Summary
The dataset contains customer-level demographic, financial, and lifestyle information related to bike purchases.

### Columns:
| Field | Description |
|--------|-------------|
| **Marital Status** | Customer’s marital status (Married/Single) |
| **Gender** | Male or Female |
| **Income** | Annual income of the customer |
| **Children** | Number of children |
| **Education** | Highest education level attained |
| **Occupation** | Type of job (Professional, Clerical, Skilled Manual, etc.) |
| **Home Owner** | Whether the customer owns a home |
| **Cars** | Number of cars owned |
| **Commute Distance** | Distance traveled to work (e.g., 0–1 Miles, 2–5 Miles) |
| **Region** | Customer’s region (Europe, North America, Pacific) |
| **Age** | Customer’s age |
| **Age Bracket** | Categorized age group (Adolescent, Middle Age, Old) |
| **Purchased Bike** | Indicates if the customer bought a bike (Yes/No) |

## 📈 Dashboard Insights
The Excel dashboard provides an interactive visualization environment where users can explore data through slicers and charts.

### 🔹 Income vs. Bike Purchase
- Customers with **lower to mid-level income** are more likely to purchase bikes.  
- Non-buyers typically have higher average income across both genders.

### 🔹 Age Bracket Trends
- **Middle-aged** customers dominate bike purchases.  
- Both **adolescents** and **older** customers show lower buying interest.

### 🔹 Commute Distance
- Short-distance commuters (**0–1 miles**) represent the majority of buyers.  
- Customers with longer commutes (**10+ miles**) are least likely to purchase bikes.

### 🔹 Regional Comparison
- The **Pacific region** shows stronger bike sales performance compared to **Europe** and **North America**.
